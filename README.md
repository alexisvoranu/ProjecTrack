ProjecTrack, my ASP.NET Core app, linked to Microsoft SQL Server, enables project managers register, manage multiple projects, and assign team members based on availability. For each project, managers can add activities and automatically calculate the critical path, which can be displayed graphically and downloaded as a PNG file.
Team members have their own access, allowing them to view assigned projects and critical path estimates. They can update the status of their tasks when they begin or complete them, ensuring real-time progress tracking and the system notifies project managers via email whenever a task status is updated, keeping everyone informed and the project on track. This integrated approach streamlines project management, enhances team collaboration, and ensures timely updates and efficient project execution.
