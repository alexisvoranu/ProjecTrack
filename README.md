ProjecTrack, my ASP.NET Core app, linked to Microsoft SQL Server, enables project managers to optimize schedules by adding activities and calculating the critical path, which can also be visually estimated and downloaded as a PNG chart. Project managers can assign all available team members to specific projects.
The other type of users are project team members, which can view the projects they are assigned to, see the critical path estimates, update the status of activities, and the responsible manager is then notified via email.
